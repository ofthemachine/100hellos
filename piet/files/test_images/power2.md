# power2.png

## Description
Piet program that calculates powers (exponentiation)

## Usage
```bash
echo "12 3" | ./piet test_images/power2.png 1
```

## Parameters
- **Image**: `test_images/power2.png`
- **Codel size**: `1`
- **Input**: Two integers: base and exponent (e.g., "12 3")

## Expected Output
```
1728
```
(12³ = 1,728)

## Notes
- Mathematical operation: calculates base^exponent
- Uses codel size 1 (single pixel codels)
- Demonstrates arithmetic and loops in Piet
- Part of the original frank-zago test suite
- Example: 12³ = 12 × 12 × 12 = 1,728
