# piet_factorial.png

## Description
Piet program that calculates the factorial of an input number

## Usage
```bash
echo "12" | ./piet test_images/piet_factorial.png 1
```

## Parameters
- **Image**: `test_images/piet_factorial.png`
- **Codel size**: `1`
- **Input**: Integer number (e.g., 12)

## Expected Output
```
479001600
```
(12! = 479,001,600)

## Notes
- Mathematical program that calculates factorials
- Uses codel size 1 (single pixel codels)
- Demonstrates loops and arithmetic operations in Piet
- Part of the original frank-zago test suite
- Example shows factorial of 12 = 479,001,600
