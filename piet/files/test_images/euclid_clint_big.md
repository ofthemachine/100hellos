# euclid_clint_big.png

## Description
Piet program that implements Euclid's algorithm for finding GCD (Greatest Common Divisor)

## Usage
```bash
echo "10 6" | ./piet test_images/euclid_clint_big.png 10
```

## Parameters
- **Image**: `test_images/euclid_clint_big.png`
- **Codel size**: `10`
- **Input**: Two integers separated by space (e.g., "10 6")

## Expected Output
```
2
```
(GCD of 10 and 6 is 2)

## Notes
- Mathematical algorithm implementation in Piet
- Uses codel size 10, indicating 10x10 pixel blocks
- Demonstrates advanced programming concepts in Piet
- Part of the original frank-zago test suite
- Example: GCD(10, 6) = 2
