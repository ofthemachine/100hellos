# hi.png

## Description
Simple Piet program that outputs "Hi"

## Usage
```bash
./piet test_images/hi.png 16
```

## Parameters
- **Image**: `test_images/hi.png`
- **Codel size**: `16`

## Expected Output
```
Hi
```

## Notes
- This is one of the working test images from the original frank-zago repository
- Uses codel size 16, indicating the image has blocks of 16x16 pixels
- Simple, reliable program that demonstrates basic Piet execution
