# progopedia_hello_basic.png

## Description
Basic "Hello, World!" program from Progopedia

## Usage
```bash
./piet test_images/progopedia_hello_basic.png 1
```

## Parameters
- **Image**: `test_images/progopedia_hello_basic.png`
- **Codel size**: `1`

## Expected Output
```
Hello, World!
```

## Notes
- Simple, reliable Hello World implementation
- Uses codel size 1 (single pixel codels)
- Part of the additional hello world test suite
- Good example of clean, minimal Piet programming
