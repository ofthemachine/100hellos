# tetris.png

## Description
Piet program that outputs "Tetris"

## Usage
```bash
./piet test_images/tetris.png 1
```

## Parameters
- **Image**: `test_images/tetris.png`
- **Codel size**: `1`

## Expected Output
```
Tetris
```

## Notes
- Uses codel size 1 (single pixel codels)
- Part of both quick-test and test-original suites
- Demonstrates text output in Piet
- One of the working test images from the original frank-zago repository
